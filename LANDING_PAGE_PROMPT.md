# Prompt para Criação da Landing Page de Vendas

**Contexto:**
Você é um especialista em UI/UX e Copywriting focado em conversão. Seu objetivo é criar o código de uma Landing Page de alta performance para o aplicativo **"Gerente de Bolso"**.

**Sobre o Produto:**
O "Gerente de Bolso" é um assistente financeiro pessoal e gerenciador de negócios para autônomos e freelancers. Ele ajuda a organizar clientes, agendamentos e finanças em um só lugar, com ajuda de IA.

**Stack Tecnológica:**
- React (Vite)
- Tailwind CSS (Estilização moderna e responsiva)
- Lucide React (Ícones)
- Design Mobile-First

**Estrutura da Landing Page (Seções):**

1.  **Hero Section (Dobra Principal):**
    *   **Headline:** "Deixe a bagunça para trás. Organize seu negócio em um toque." (Ou algo mais impactante focado em liberdade/organização).
    *   **Subheadline:** "O assistente completo para gerenciar clientes, agendamentos e seu dinheiro. Simples, rápido e inteligente."
    *   **CTA Principal:** "Começar Gratuitamente" (Botão de destaque).
    *   **Visual:** Uma imagem de alta qualidade ou mockup do app mostrando a dashboard organizada.

2.  **Problema (A Dor):**
    *   Foque nas dores do freelancer: "Cansado de planilhas complexas?", "Esqueceu de cobrar um cliente?", "Agenda bagunçada?".
    *   Use ícones e textos curtos para fácil leitura.

3.  **Solução (Benefícios):**
    *   **Gestão de Clientes:** Todo o histórico e contatos na palma da mão.
    *   **Agendamento Inteligente:** Nunca mais perca um horário.
    *   **Controle Financeiro:** Saiba exatamente quanto entra e sai.
    *   **Assistente IA:** Destaque o diferencial da IA que ajuda a tomar decisões.

4.  **Prova Social (Depoimentos):**
    *   3 cards com depoimentos fictícios (mas realistas) de usuários satisfeitos (ex: "Mudou minha vida como designer freelancer").

5.  **CTA Secundário (Middle):**
    *   Uma chamada para ação no meio da página para quem já se convenceu.

6.  **FAQ (Perguntas Frequentes):**
    *   "É gratuito?"
    *   "Funciona offline?"
    *   "Meus dados estão seguros?"

7.  **Footer:**
    *   Links úteis, contato e novamente o CTA final.

**Diretrizes de Design:**
- **Estilo:** Clean, Moderno, "Premium". Use bastante espaço em branco (whitespace).
- **Cores:** Use a paleta do projeto (Azul brand: #0ea5e9).
- **Tipografia:** Sans-serif moderna, fácil leitura.
- **Responsividade:** Deve ficar perfeito no celular.

**Instruções de Código:**
- Gere o código em um único arquivo ou componentes bem separados se necessário.
- Use `framer-motion` se possível para animações suaves de entrada (fade-in, slide-up).
- Garanta que todos os botões tenham estados de `hover` e `active`.
